Love programming but hate writing docs?

Same here. That's why I built **docs-gen** — a tool that makes it as easy as possible to spin up a documentation site. Just write Markdown files, run a single command, and you've got a site like this. Deploying to GitHub Pages is a breeze too.

![Screenshot](./screenshot.png#large)
